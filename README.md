# 🌾 KrishiMitra - AI Powered Agriculture Advisory Platform

An AI-powered agriculture advisory platform designed to help farmers make informed decisions through crop recommendations, disease detection, weather forecasting, market insights, and an intelligent multilingual chatbot.

---

## 📌 Overview

KrishiMitra empowers farmers by combining Artificial Intelligence, Machine Learning, and modern web technologies into a single platform.

The application provides personalized agricultural guidance, helping farmers improve productivity and reduce losses.

---

## ✨ Features

### 🤖 AI Chatbot
- AI-powered farming assistant
- Multilingual support
- Instant agricultural guidance
- Natural language conversations

### 🌱 Crop Recommendation
- Suggests suitable crops based on:
  - Soil Type
  - Soil pH
  - Nitrogen (N)
  - Phosphorus (P)
  - Potassium (K)
  - Previous Crop

### 🦠 Plant Disease Detection
- Upload crop images
- AI detects plant diseases
- Provides treatment suggestions
- Preventive recommendations

### 🌦 Weather Forecast
- Real-time weather information
- Temperature
- Humidity
- Rain prediction
- Farming recommendations

### 💹 Market Price Analysis
- Current crop prices
- Market trends
- Price comparison
- Better selling decisions

### 🏛 Government Schemes
- Agricultural schemes
- Subsidies
- Loan information
- Farmer welfare programs

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Shrisai2005/Krishimitra_official.git
```

### Navigate

```bash
cd Krishimitra_official
```

### Install Backend Dependencies

```bash
pip install -r requirements.txt
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Run Backend

```bash
uvicorn main:app --reload
```

### Run Frontend

```bash
npm start
```
