# 🌾 KrishiMitra - AI Powered Agriculture Advisory Platform

An AI-powered agriculture advisory platform designed to help farmers make informed decisions through crop recommendations, disease detection, weather forecasting, market insights, and an intelligent multilingual chatbot.

---

## 📌 Overview

KrishiMitra empowers farmers by combining Artificial Intelligence, Machine Learning, and modern web technologies into a single platform.

The application provides personalized agricultural guidance, helping farmers improve productivity and reduce losses.

---

### 🤖 AI Chatbot

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Shrisai2005/Krishimitra_official.git
```

### Navigate

```bash
cd Krishimitra_official
```

### Install Backend Dependencies

```bash
pip install -r requirements.txt
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Run Backend

```bash
uvicorn main:app --reload
```

### Run Frontend

```bash
npm start
```

# 🌾 KrishiMitra – Smart Agriculture Assistant

KrishiMitra is an AI-powered smart agriculture advisory platform designed to help farmers make better decisions using real-time insights and intelligent recommendations.

It supports farmers throughout the entire crop lifecycle — from crop selection to disease detection, weather monitoring, and market selling decisions.

---

## 🚀 Features

### 💬 Multilingual AI Chatbot
- Ask farming questions in Kannada, Hindi, English
- Voice input & output support
- Provides simple, actionable advice

### 📸 Disease Detection (AI + ML)
- Upload crop image
- Detect disease + severity level
- Get treatment & prevention steps

### 🌦️ Weather Forecast
- 5-day weather prediction
- Smart farming advice (irrigation, spraying, etc.)

### 🏦 Smart Schemes & Loans
- AI-based personalized recommendations
- Government schemes + bank loans
- Direct official links

### 📈 Market Intelligence
- Crop price analysis by district/taluka
- Best mandi to sell
- Price trend + prediction graph
- “Sell Now vs Wait” decision
